Weather Site Name

Local Weather Watchers

Site Purpose

The purpose of this site is to provide up-to-date weather forecasting in the local area of anyone who uses this site. It will provide interactive icons that will display the current weather as well as the forecasted weaather for the foreseeable week. This site will be accessible on any viewports for easy access to viewers who use laptops as well as smartphones.

Domain Name

"localweatherwatchers.com" per availability at hostgator.com for $12.95/year.
